### Implementation Details

This was our very first Pull Request, laying the visual foundation for the entire project! 

The core challenge was getting the **Fractal Brownian Motion (FBM)** to look organic rather than strictly mathematical. We achieved this by rotating the noise coordinates slightly at each octave to reduce axial bias:

```glsl
// Rotate to reduce axial bias
mat2 rot = mat2(cos(0.5), sin(0.5), -sin(0.5), cos(0.50));
for (int i = 0; i < 5; ++i) {
  v += a * cnoise(x);
  x = rot * x * 2.0 + shift;
  a *= 0.5;
}
```

By layering multiple instances of this `fbm` function and using the output of one as the coordinate input for the next (a technique known as **Domain Warping**), we created the illusion of deep, flowing fluid without ever writing a physics solver.
